# AccountOverview (`cui-account-overview`)

<br />

## HTML Example

<br />

### Basic

AccountOverview details can be set by providing the desired details and their corresponding values.

```html
<cui-account-overview
  details='{
    "total_balance_cents": 349392,
    "credit_limit_cents": 800000,
    "available_credit_cents": 450608,
    "pending_charges_cents": 0,
    "interest_rate_percent": 8.99
  }'
>
</cui-account-overview>
```

### Custom

Each AccountOverview detail property can be configured with an object to customize the property's `label` and `value`.
Leaving out the `label` property sets the label name to its default value.

```html
<cui-account-overview
  details='{
    "total_balance_cents": {
      label: "Balance",
      value: 349392,
    },
    "credit_limit_cents": {
      label: "Limit",
      value: 800000,
    },
    "available_credit_cents": 450608,
    "pending_charges_cents": 0,
    "interest_balance_cents": 1198,
    "interest_rate_percent": 8.99,
  }'
>
</cui-account-overview>
```

## JavaScript Example

<br />

### Basic

AccountOverview details can be set by providing the desired details and their corresponding values.

```html
<div id="account-overview-container"></div>

<script>
  const accountOverviewEl = document.createElement("cui-account-overview");
  const accountDetails = {
    total_balance_cents: 349392,
    credit_limit_cents: 800000,
    available_credit_cents: 450608,
    pending_charges_cents: 0,
    interest_rate_percent: 8.99,
  };
  accountOverviewEl.setAttribute("details", JSON.stringify(accountDetails));
  document.getElementById("account-overview-container").appendChild(accountOverviewEl);
</script>
```

Leaving out the `label` property sets the label name to its default value.
